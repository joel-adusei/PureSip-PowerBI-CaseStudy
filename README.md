# PureSip-PowerBI-CaseStudy

📊 𝗣𝗼𝘄𝗲𝗿 𝗕𝗜 𝗦𝗮𝗹𝗲𝘀 𝗔𝗻𝗮𝗹𝘆𝘀𝗶𝘀 𝗗𝗮𝘀𝗵𝗯𝗼𝗮𝗿𝗱 | 𝗣𝘂𝗿𝗲 𝗦𝗶𝗽'𝘀

    I developed an end-to-end Power BI dashboard for a beverage distribution company, PureSip Beverages, to transform their fragmented sales data into actionable business insights.

𝗕𝘂𝘀𝗶𝗻𝗲𝘀𝘀 𝗖𝗵𝗮𝗹𝗹𝗲𝗻𝗴𝗲

    Regional sales data was maintained in separate spreadsheets, making reporting time-consuming, prone to errors, and difficult to analyse at scale.

𝗦𝗼𝗹𝘂𝘁𝗶𝗼𝗻

    Using Power BI Desktop, Power Query, and Folder Connections, I built an automated reporting solution that:
     ✅ Consolidates multiple retailer spreadsheets into a single data model
     ✅ Automates data refresh when new files are added to the folder
     ✅ Reduces manual data preparation and improves data consistency
     ✅ Enables interactive analysis across retailers, brands, regions, and sales trends

𝗗𝗮𝘀𝗵𝗯𝗼𝗮𝗿𝗱 𝗛𝗶𝗴𝗵𝗹𝗶𝗴𝗵𝘁𝘀

    📈 Sales trend analysis over time
    🏪 Retailer performance comparison
    🌍 Regional sales analysis
    🥤 Beverage brand performance tracking
    📌 KPI monitoring for sales and quantity sold
